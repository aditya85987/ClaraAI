# ClaraAI

1. Go to this path: `C:\Users\yourusername`
2. Create a folder named `.n8n-files`
3. Clone this(or my) repository inside `.n8n-files`
4. Go to the `input` folder
5. Paste demo transcript in `demo.txt`
6. Paste onboarding transcript in `onboarding.txt`
7. Import `workflow.json` into n8n
8. Click **Execute Workflow**

## Required Versions
- Node.js v22.22.0
- npm v10.9.4
- n8n v2.10.3
